# smoke

A Linux privacy & anti-fingerprinting suite.

`smoke` removes, spoofs, randomizes, rotates, or conceals every hardware- and
OS-level identifier that can be used to permanently or semi-permanently
fingerprint a machine or OS install - and disrupts processes that try.

## Status

Pre-alpha. Spec and implementation plan in [`docs/`](./docs). Nothing ships yet.
